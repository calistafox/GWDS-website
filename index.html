Coding Worksheets

The coding worksheets begun at the very basics, first taking us through creating a pure HTML website, learning about adding content and seeing how the different elements react before we add our owns styling. We also set up GitHub and looked into the workflow of storing your code online. In the following weeks we used CSS to style the webpage, adding colours, styling elements, defining fonts, CSS selectors and used flexbox to organise content. The final project developed my skills in flexbox, first using it to make the navigation appear horizontally, with some aligned to the left and some to the right margin. Also using it to centre content and align it next to each other. In next weeks worksheet we styled it further, making a button link to an external html document, animating buttons and imported google and adobe fonts. Lastly, I completed the optional carousel, where I styled 5 images be in a horizonal row using flex, which can be navigated through like a slideshow using a scroll bar. After completing these worksheets, I now have a good understanding of how a website is built and look forward to applying these skills for my final website.
